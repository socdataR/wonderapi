D76 Detailed Mortality Dataset Codebook

Joyce Robbins

2017-08-28

This codebook is provided to assist in determining how to set up a query to the CDC Wonder API. The best way to become familiar with the API is to use the web interface: https://wonder.cdc.gov. Also, read the Introduction to wonderapi to learn about the basic sections of CDC Wonder query requests.
